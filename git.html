git config // Просмотр настроенных опций git help // Просмотр настроенных опций git -h // Просмотр настроенных опций git init // После установки создание рабочей директории git clone {url} {localName} // Клонирует директорию по указонному {url}. {localName} - желаемое имя директории git status // Статус файлов git status -s // Статус файлов упрощенная версия git add {name} // Добавить в индекс файл/директорию git diff // Просмотр изменений в файлах. Имеет много параметров для фильтрации вывода данных git diff --staged // Просмотр изменений в непроиндексированных файлах git diff --cached // Просмотр изменений в проиндексированных файлах git commit // Создать коммит git commit -v // Создать коммит с данными {diff} git commit -m // Создать коммит с сообщением git commit -a // Создать коммит и добавить в него все файлы git commit --amend // Отменить последний коммит git rm -f {name} // Удалить файл с локального репозитория git rm --cached {name} // Удалить файл из индексирования git rm --cached -r {./directory} // Удалить директорию из индексирования рекурсивно git log // Просмотр истории коммитов для всех веток. Имеет много параметров для фильтрации вывода данных git checkout {branch} // Переключиться на ветку {branch} git checkout -b {branch} {origin}/{branch} // Создать ветку, переключиться на нее и слить данные ветки {branch} с удаленного репозитория {origin} git checkout --track {origin}/{branch} // Создать ветку, переключиться на нее и слить данные ветки {branch} с удаленного репозитория {origin} git checkout -b {branch} // Создать и переключиться на ветку {branch} git remote // Просмотр удаленных репозиториев git remote -v // Просмотр добавленных удаленных репозиториев более подробно git remote add {origin} {url} // Добавление репозитория {url} к себе под именем {origin} git remote show {origin} // Показать всю информацию о ветках удаленного и локального репозитория git fetch {origin} // Получает данные с удаленного репозитория git fetch {origin} {branch} // Получает данные с удаленного репозитория только для ветки {branch} git pull {origin} // Объединить все ветки с удаленного репозитория {origin} со своими локальными git pull {origin} {branch} // Объединить ветку {branch} c веткой локального репозитория git pull {origin} HEAD // Объединить текущую ветку c веткой локального репозитория git push -u {origin} {branch} // Отправить ветку на удаленный репозиторий и настоить слежение за этой веткой git push {origin} {branch} // Отправить на удаленный репозиторий {origin} ветку {branch} git push {origin} HEAD // Отправить на удаленный репозиторий текущую ветку git config --global alias.{name} {command} // Изменения или создание новой {name} команды {command} git git branch // Посмотреть список всех локальных веток git branch --list // Посмотреть список всех локальных веток git branch --show // Посмотреть текущую ветку (только название) git branch --list {pattern} // Посмотреть список всех локальных веток, используя {pattern} в поиске git branch -r // Посмотреть список всех удаленных веток git branch -a // Посмотреть список всех локальных и удаленных веток git branch -v // Посмотреть хеш и сообщение коммита локальных веток git branch -vv // Посмотреть хеш, слежение и сообщение коммита локальных веток git branch --merged // Посмотреть ветки, для которых уже производилось слияние (можно удалить) git branch --no-merged // Посмотреть ветки, для которых не производилось слияние git branch {name} // Создать новую ветку с именем {name} git branch -d {name} // Удалить ветку {name}. Удаление произойдет если ветка не содержит коммитов git branch -D {name} // Удалить ветку {name}. Удаление произойдет в любом случае, даже если есть коммиты git push {origin} -d {branch} // Удалить отслеживаемую ветку {branch} с удаленного репозитория {origin}. git show {commit} // Посмотреть изменения коммита {commit} git switch {branch} // Переключиться на ветку {branch} git revert HEAD // Отмена комита, на котором расположен HEAD, создает новый комит отмены git reset --hard {hash} // Удаляет всю историю комитов до указанного {hash} в текущей ветке git reset HEAD {name} // Исключает файл из индекса git merge {branch} // Произвести слияние текущей ветки с веткой {name} git mergetool // Посмотреть подробную информацию о конфликте слияния
